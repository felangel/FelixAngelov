Rebus Puzzle

February 2nd, 2016

This is my rebus puzzle. I decided to try to illustrate the saying Every cloud has a silver lining because I feel like the saying itself is very inspirational partly because it can be interpreted in numerous ways. As a result, it seemed only fitting to use this phrase for my rebus. I used Eve from the Bible along with Rye to make Every. Then I drew a simple cloud using the pen tool followed by the letter H and an ass to make the word has. I used the letter A along with the element from the periodic table for silver and concluded with the mathematical equation of a line with the letters ing. This piece allowed me to continue to practice using the pen tool as well as converting drawings into smart images which I could fill using the smart fill tool. I also played a bit with the typography and fill of the letters to supplement the images.
